# Competitive Gap Analysis

## Hypothesis

A **self-hosted**, audit-first console that fuses stealth crawl, ethical OSINT, agentic missions, and geospatial fusion will out-position single-purpose SaaS scrapers and aging OSINT tools for enterprise SOC / red-team / gov-adjacent buyers who care about data residency and ROE.

## Evidence-based gaps

### Firecrawl
- Optimized as an API for AI apps (scrape/crawl/map).
- Strong LLM-ready markdown — HelixaraAI matches this extraction goal.
- Lacks: multi-module command HUD, engagement attestation, mission swarm, geospatial correlation, sovereign default posture.

### Classic OSINT suites
- Broad source coverage but weak modern stealth crawl + agent orchestration UX.
- Compliance/audit often bolted on rather than primary design.

### Closed fusion platforms
- High capability, high cost/lock-in.
- HelixaraAI targets the sovereign mid-market and internal red-team labs.

## Measurable KPIs (MVP targets)

| KPI | Target |
|-----|--------|
| Scrape success on public static sites | ≥95% |
| Guardrail block rate on private IPs | 100% |
| Audit completeness on sensitive ops | 100% |
| OSINT domain enrichment time | <15s typical |
| Agent mission multi-step completion | ≥85% on public targets |

## Positioning statement

> **HelixaraAI** is the self-hosted command layer for authorized cyber intelligence: stealth crawl that respects ROE, OSINT that stays ethical, agents that report — not weaponize — and a geospatial HUD that turns findings into situational awareness.
